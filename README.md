# Gravity Pro 🚀

**Gravity Pro** is a next-generation UPI Escrow Freelance Marketplace MVP. It connects vetted enterprise-grade freelance talent with clients under secure, milestone-based escrow contracts verified instantly via UPI QR code simulation.

## 🔗 Live Site
Go directly to the public website here:
👉 **[https://gravity-pro-escrow.vercel.app](https://gravity-pro-escrow.vercel.app)**

---

## 🌟 Key Features
- **Secure UPI Escrow Pool**: Milestone funds are safely held in escrow and only released to the freelancer's VPA after client verification.
- **Dynamic QR Code Billing**: Instant verification via simulated webhook scan flows.
- **Milestone & Delivery Tracking**: Client-Freelancer collaborative workspaces with real-time chat, file transfer logs, and release workflows.
- **Premium Dark/Light Theme**: Sleek dark and light designs using modular CSS variables and preserved state preferences.
- **Optimized Assets**: Beautiful WebP AI-generated graphics with interactive micro-animations.

---

## 🛠️ Tech Stack & Structure
- **Frontend**: React + Vite (Javascript)
- **Styling**: Vanilla CSS with custom design variables
- **Icons**: Lucide React
- **Celebrations**: Canvas Confetti
- **Deployments**: Vercel

## 📦 Getting Started Locally

1. **Clone the repository**:
   ```bash
   git clone https://github.com/keyz-loop/gravity-pro-escrow.git
   cd gravity-pro-escrow
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the local development server**:
   ```bash
   npm run dev
   ```

4. **Build the production bundle**:
   ```bash
   npm run build
   ```
